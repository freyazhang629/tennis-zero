## 🔗 连接GitHub仓库

1. **访问GitHub**: https://github.com/new
2. **仓库名称**: `tennis-zero`
3. **设置**: Public, 不要勾选任何初始化选项
4. **创建仓库**

然后运行：
```bash

git remote add origin https://github.com/YOUR_USERNAME/tennis-zero.git
git push -u origin main
```
